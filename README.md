  ## About The Project
  
Projet Designé par <strong>Michael Yeates</strong>, traduit et adapté en français par Mahmoud Ismail

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mdyeates/my-portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
